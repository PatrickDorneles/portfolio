# ADR Template

Use this template to document architectural decisions.

## [ADR-XXX]: Title

**Status**: Proposed / Accepted / Superseded
**Date**: YYYY-MM-DD
**Decided by**: [Agent/User]

### Context
Describe the problem and the alternatives considered. Why is a change needed?

### Decision
What was decided? Which option was chosen and why?

### Consequences
What are the positive and negative impacts of this decision?

### Verification
How do we know this decision was implemented correctly?
